阿里云建立微信公众号的相关资料

1、由于近期发现itchat开源项目要升级到itchatmp , 所以考虑下把自己的阿里云ECS利用起来，不然有点浪费了。

2、首先熟悉下flask的hello world实现：

#encoding:utf8

from flask import Flask


app = Flask(__name__)
@app.route('/')
def hello_world():
    return 'Hello World!'

if __name__ == '__main__':
    app.run(host='127.0.0.1', port=9527,debug=True)

其中：@app.route(‘/‘) 是定义的路由，表示对应网址的根目录。
要注意的是，这段简单的代码里：app.run(host=’127.0.0.1’, port=9527,debug=True)
如果把127.0.0.1 修改成 0.0.0.0 则报错。 原因暂时不确定。
