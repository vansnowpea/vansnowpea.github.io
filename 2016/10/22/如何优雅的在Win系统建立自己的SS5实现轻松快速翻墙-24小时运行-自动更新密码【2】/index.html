如何优雅的在Win系统建立自己的SS5实现轻松快速翻墙【2】-24小时运行-自动更新密码

Veröffentlicht am 2016-10-22

神之力量击穿叹息之墙

1、首先，请参考我的上一个帖子，实现了Win下的SS【快速】翻墙，如果没有SS，要翻墙都要忍受蜗牛速度。为SS的原版牛逼作者十万个赞，也要为翻版作者以及iss网站一万个赞。但是有个小问题，因为源头的SS账号每隔6小时变更，因此总是需要人工更新密码，那么如何把他做成自动更新？

2、方案，爬虫捕捉新密码+重写json加载。

3、细节分析：

谷歌已经有人写了相关代码，可能是对照原版SS写的，如今我是用的是替代版本，因此需要做一定修改，【必须要谷歌*3】

另外，我是用lxml代替了bs4，个人喜好。数据部分也变更修改，可能原作者写的时候网站数据模式和如今的不同，另增加了一些函数，而json部分的代码保留，只需根据自己的文件路径修改即可。

如下图，因为iss变更的是密码，因此爬虫也抓密码就好，比较简单

4、代码,关于json部分请看参考资料部分：

# -*- coding: utf-8 -*-
# python 3.5.2
# 测试系统，Win10
# Author:Van
# 实现SS5本机自动更新密码，争取24小时保持连接
# 实际做不到24小时，因为iss网站的密码的变更有一定时间的延迟
# 所以最好能有其他的SS源头网站
# V1.0
# 欢迎各种改进意见
# 请把对应的帐号密码修改成自己的
# 请把SS软件的路径改成自己的


import requests, json, os
from lxml import etree
import smtplib
from email.mime.text import MIMEText
from email.header import Header
import copy
import time

# define the time gap for checking in seconds
# 60 equals to 1 minute
# 3600 equals to 1 hour
time_gap = 300
url = 'http://www.ishadowsocks.org'
html = requests.get(url).content.decode('utf-8')
selector = etree.HTML(html)
password = selector.xpath('//*[@id="free"]/div/div[2]//h4[3]/text()')

print('实时密码是：', password)
# temp_password = copy.deepcopy(password)
temp_password = []
print('临时密码是：', temp_password)
a_pass = password[0].split(':')[-1]
b_pass = password[1].split(':')[-1]
c_pass = password[2].split(':')[-1]

# 打开配置json文件
# indent参数是缩进的意思
class JSONObject:
    def __init__(self,d):
        self.__dict__=d
    def to_JSON(self):
        return json.dumps(self, default=lambda o: o.__dict__,
            sort_keys=True, indent=4)

def modify_json():
    with open('E:\\迅雷下载\\ShadowsocksR-win-3.7.4.1\\gui-config.json','r') as f:
        data = json.load(f,object_hook=JSONObject)
        data.configs[0].password=a_pass
        data.configs[1].password=b_pass
        data.configs[2].password=c_pass
        data=data.to_JSON()
    with open('E:\\迅雷下载\\ShadowsocksR-win-3.7.4.1\\gui-config.json','w') as of:
        of.write(data)
    #打开程序
    # os.system('E:\\迅雷下载\\ShadowsocksR-win-3.7.4.1\\ShadowsocksR-dotnet4.0.exe')

def check_update():
    # global password
    password = selector.xpath('//*[@id="free"]/div/div[2]//h4[3]/text()')
    # print(password)
    if temp_password == password:
        return password,False
    elif temp_password != password:
        print('password changed, will update!')
        return password,True



def send_email(htm):
    # send email to notice new WestWorld is coming
    sender = 'fanqikai@163.com'
    receiver = '409085788@qq.com,fanqikai@163.com'
    subject = 'SS5密码有更新！'
    smtpserver = 'smtp.163.com'
    username = 'fanqikai@163.com'
    password = 'fanka111'
    msg = MIMEText(htm, 'html', 'utf-8')
    msg['Subject'] = Header(subject, 'UTF-8')
    msg['From'] = sender
    msg['To'] = ','.join(receiver)
    smtp = smtplib.SMTP()
    smtp.connect(smtpserver)
    smtp.login(username, password)
    smtp.sendmail(sender, receiver, msg.as_string())
    smtp.quit()

if __name__ == '__main__':
    password, update = check_update()
    while 1:
        if update :
            modify_json()
            try:
                send_email(str(password))
            except Exception as e:
                print("email failed:", e)
        temp_password = copy.deepcopy(password)
        print('新的临时密码是：', temp_password)
        time.sleep(time_gap)
